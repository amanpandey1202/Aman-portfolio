## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/amanpandey1202/Aman-portfolio.git
```

### Navigate to the project

```bash
cd Aman-portfolio
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The application will run locally at:

```text
http://localhost:5173
```
